# TileFix

A small graphics mod for Duke Nukem 3D, aiming to beautify and enhance the game. The project has entailed cleaning up dirty coloring/stray pixels, extending screen tiles horizontally for widescreen compatibility, extending some screen tiles vertically to provide extra cover for screenbob, creating missing animation frames for back rotations, fixing some animation and offset errors that don't require CON scripting, etc. A bonus feature is the restoration of an unused item, AMMOLOTS (tile #48), which had no art but complete programming. I do not believe this item has been placed anywhere in the game, and it is unlikely any usermaps have used it either. The item will give the player 48 pistol ammo. Two bonus textures have been included replacing duplicate tiles (#4321 \& #4322), the former of which is unused in the base game and the latter is an alternate version of the dupe.



This mod is free to use for any purpose, including use in mods for Duke Nukem 3D or other games. I only ask that you give credit where it is due.

# Construction

Editors Used: Adobe Photoshop CS6, MeeSoft's Image Analyzer, IrfanView, LibreSprite, BAFEd, EditArt, Magica Voxel

Construction Time: TileFix officially began development in April of 2025. Some of these tiles I have edited in the past ten years or so.  
Known Bugs/Problems: May introduce graphical issues with user maps that contain creative texture or sprite work. It should not introduce issues to the base game.



# Installation

Extract the TileFix folder into your base EDuke32 folder, and select TileFix for custom game content directory. More compatibility will be added for addons and other mods in the future.



# Notes

Interestingly, during the course of doing this I found that some textures from LameDuke actually had better color depth than textures that ended up in the final game. Retaining or increasing color depth in all edits was a high priority. Some small liberties have been taken here or there, mostly logic fixes that fit within the scope of the game's vision. For some tiles, there are optional alternate versions closer to the original art or extraneous edits that would be useful to mappers/modders. Included are many .PSDs mostly for those that are curious or "exploitables" so that it is easier for others to create edits. Many of these exploitables saved me time fixing multiple similar tiles.
